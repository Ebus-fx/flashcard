Are you new to Javascript? here are some basic things you need to know as a beginner

  • JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.
  • JavaScript is a scripting language for creating dynamic web page content.

  • jQuery is an open-sourced JavaScript library that simplifies creation and navigation of web applications.
  • jQuery simplifies HTML Document Object Model (DOM) manipulation, Asynchronous JavaScript and XML (Ajax) and event handling.

  • An operator, in Java, is a special symbols performing specific operations on one, two or three operands and then returning a result..
  • Operators are generally used to manipulate primitive data types.

  • A string is a sequence of one or more characters that may consist of letters, numbers, or symbols.
  • Strings in JavaScript are primitive data types and immutable, which means they are unchanging.

  • a boolean value is one that can either be TRUE or FALSE. If you need to know “yes” or “no” about something, then you would want to use the boolean function.
  • booleans are used all the time in JavaScript programming, and they are extremely useful.

  • A variable that has not been assigned a value is of type undefined.
  • A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.

  • NaN is short for "Not-a-Number".
  • NaN is a number that is not a legal number.

  • null is a special value that represents the intentional absence of any object value.
  • It is often used as a placeholder or to indicate that a variable has no value or an empty value. When a variable is assigned with null , it means that the variable has no value currently.

  • A function in JavaScript is similar to a procedure—a set of statements that performs a task or calculates a value, but for a procedure to qualify as a function
  • it should take some input and return an output where there is some obvious relationship between the input and the output.

  • parameters are variables that you list as part of a function definition.
  • They act as placeholders for values that will be passed to the function when it is called.

  • operators are special symbols used in scripts to perform operations on operands, such as arithmetic calculations, logical comparisons, or value assignments. operands and then returning a result..
  • Operators are generally used to manipulate primitive data types. It plays a crucial role in controlling the flow and processing of data within the language.

  • arguments is an array-like object accessible inside functions that contains the values of the arguments passed to that function.
  • Arguments is an object that is accessible inside functions that contain the values of the arguments passed to that function.
